This project implements the Simplex algorithm in C++ to solve a linear programming problem (minimizing cost with constraints).
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=--=-=-=-=-=-=-=-=-=-=-=-==-=-=-=-=-=-=--=-=--=-=-=-=-=-=-=-=-=-=-=-==-=-=-=-=-=-
The Files Are 
	1) main.cpp: Runs the Simplex algorithm on a sample problem.
	2) Q2.h: Contains the Simplex class declaration.
	3) Q2.cpp: Contains function definitions for the Simplex class.
	
To Run the Code
	1) Run "Make"
	2) ./main
	3) Output is presented
	4) "Make Clean" to remove executables
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=--=-=-=-=-=-=-=-=-=-=-=-==-=-=-=-=-=-=--=-=--=-=-=-=-=-=-=-=-=-=-=-==-=-=-=-=-=-	
The Simplex algorithm has a worst-case time complexity of exponential order, specifically O(2^n) for some problem instances.However, it is commonly fast and reliable, making it a popular choice for solving linear programming problems.
